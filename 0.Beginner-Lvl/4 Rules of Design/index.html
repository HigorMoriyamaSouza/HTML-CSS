4 Rules of Design

Spacing

It's a simple rule that gets broken all the time. You need to group similar elements together and then enclose them with empty space. Spacing can be a surprisingly powerful tool.

Grade

The dimensions of the elements and the distances between them are organized in a cohesive structure known as a grid. If an element falls outside the grid, you'll know something is wrong.

Fonts

According to this rule, there should be no more than three different font types on a single website. One for the header, one for the body text, and a third to highlight essential words.

Colors

In addition to the main color, there are also accent colors used for important details on a website. Use the right tools to select a good color combination.
